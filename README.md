# Generalized Decidability via Brouwer Trees

This repository contains a formalization of the constructions and proofs of the paper _Generalized Decidability via Brouwer Trees_.

It uses (Cubical) Agda 2.8.0 and the Cubical library (commit: fe1bd60a).

The files directly relevant to the paper are in [BrouwerTree/OrdinalDecidability/](BrouwerTree/OrdinalDecidability/), see [Paper.agda](BrouwerTree/OrdinalDecidability/Paper.agda) in particular.
